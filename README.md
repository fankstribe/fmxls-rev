# FmxlsRev 

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 10.1.7.

## Requisites

* git
* node.js
* npm
* mongodb

## Installation

```
git clone https://github.com/fankstribe/fmxls-rev.git
cd fmxls-rev 
npm install 
cd backend
npm install
cd ..
npm run dev
```

## Update

```
cd fmxls-rev
git pull
npm install
cd backend
npm install
cd ..
npm run dev
```
